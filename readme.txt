A Simple plugin to Update Comment Count

== Description ==
This plugin Simpliy Update the Comment Count in a one click of button , it is used when you have redisgn your website and moved the comments from 
one website to another at that time it will help you to recalculate the comment count. 


==Features==
1. Update Comment Count

== Installation ==

1. Upload `update-comment-count` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
